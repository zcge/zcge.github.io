Markdown

markdown 是一种轻量级的「标记语言」

优点：

  • 导出 HTML,PDF文件
  • 专心你的文字而不必在意你的排版

工具推荐:

  • 在 Mac OS X 上，我强烈建议你用 Mou 这款免费且十分好用的 Markdown 编辑器
    • 快速导出HTML 和 PDF 文件
    • 实时预览
  • 对于那些用Window电脑的人,我推荐你先买个Mac吧。（Window是最差的开发环境）
