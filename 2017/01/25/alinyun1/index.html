阿里云 Node 安装

  • node

JAVA 太水 ,还是用node 来写后台吧（毕竟JS我还是可以的，自吹一波）

安装Node环境: 用ssh 登陆服务器

1.yum -y update 升级到最新版本

2.我们将使用最新源代码构建Node.js，要进行软件的安装，需要一组用来编译源代码的开发工具

3.cd /usr/src 切换目录到这。把软件下载这里吧。

4.选择一个稳定的node.js 版本. （其实我是不建议wget下载的。慢的一B，建议去node官方网站下载好后用ftp工具上传到服务器。也就花了我半个小时。）

node –v 看下是否成功（什么？你没成功？我只能告诉你，关我屁事。）
