nohup

  • node

问题的由来

我在终端中启动了服务node app.js

然后很开心的发现我的域名(hankewei.xyz)下可以访问网站了

然后我很开心的关掉了终端 再去访问我的网站，然后我就不开心的发现访问不到了

怎么办啊 ==> 万能的谷歌（还好厂子里的内网自带翻墙）

查资料发现可以用守护进程解决这个问题

上面这样启动的脚本，称为”前台任务”（foreground job）。它会独占命令行窗口，只有运行完了或者手动中止，才能执行其他命令。

node app.js & 这样就可以把上面的任务变成后台任务

nohup node server.js & 就是启动一个守护进程
