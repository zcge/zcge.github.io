js事件冒泡和捕获

  • 前端
  • 假设都绑定了点击事件

JS事件冒泡

  • 事件冒泡顺序是由内到外进行事件传播，直到根节点。
  • 点击p=>先触发p的事件 再触发div的事件。

JS事件捕获

  • 事件冒泡顺序是由外到内进行事件传播，直到事件源。
  • 点击p=>先触发div的事件 再触发p的事件。

如何阻止事件的传递

  • event.stopPropagation()

如何选择冒泡和事件捕获

  • addEventListener(event, listener, useCapture)方法的第三个参数 true 是捕获 false是冒泡
